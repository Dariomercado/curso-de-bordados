# UI Guidelines

## Estética

- artesanal
- cálida
- minimalista
- premium pero cercana

## Paleta base

- crema
- rosa viejo suave
- terracota clara
- verde salvia
- gris cálido

## Tipografía

- serif elegante para títulos
- sans limpia para texto

## Componentes

- bordes redondeados suaves
- sombras livianas
- spacing amplio
- botones tipo pill
- evitar apariencia SaaS

# Project Overview

## Nombre del proyecto

Atelier de Bordado

## Descripción

Sitio web para una emprendedora que vende cursos online de bordado y kits de bordado.

## Objetivo del negocio

Construir una presencia digital profesional que permita:

- presentar la marca
- vender cursos
- vender kits
- generar confianza
- escalar a futuro

## Stack elegido

- Next.js
- TypeScript
- Tailwind CSS
- App Router

## Alcance inicial

Primera fase:

- implementar la home
- usar datos mock
- crear arquitectura escalable
